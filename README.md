# 🤖 SunkenBot v2.1 — بوت فيسبوك ماسنجر

> بوت ماسنجر مكتوب بـ Node.js يعمل على حسابك كـ Userbot داخل مجموعات فيسبوك، مع دعم الذكاء الاصطناعي، تحميل الميديا، الشطرنج، القرآن الكريم، وأكثر.

---

## 📖 كيف يعمل بشكل بسيط؟

```
أنت تكتب أمر في المجموعة
        ↓
البوت يقرأ الرسالة
        ↓
يتعرف على الأمر (مثلاً: .yt اغنية)
        ↓
ينفذ الكود المناسب (مثلاً يحمل من يوتيوب)
        ↓
يرد عليك في المجموعة
```

البوت **يتظاهر أنه أنت** — يسجل دخول لحساب فيسبوك حقيقي ويرسل ردود باسمه. لذا تحتاج حساباً مخصصاً للبوت.

## 🚀 خطوات التشغيل

### 1. تثبيت المكتبات

```bash
npm install
```

### 2. إنشاء ملف `.env`

أنشئ ملف `.env` في جذر المشروع وضع فيه:

```env
# بيانات حساب فيسبوك البوت
FB_EMAIL=email@example.com
FB_PASSWORD=yourpassword

# إن كان الحساب يستخدم 2FA (اختياري)
FB_2FA_SECRET=

# قاعدة البيانات (اختياري — البوت يشتغل بدونها)
MONGO_URI=mongodb+srv://...

# مفاتيح API للأوامر الذكية (حسب ما تريد تفعيله)
CEREBRAS_API_KEY=
HF_SPACE_URL=
HF_SCRAPER_URL=
```

### 3. إعداد `config.json`

```json
{
  "admins": ["ID_حسابك_على_فيسبوك"],
  "moderators": [],
  "developers": [],
  "vips": [],
  "Prefix": ["."],
  "botName": "SunkenBot"
}
```

> **كيف أعرف ID حسابي؟** استخدم أمر `.uid` داخل المجموعة بعد تشغيل البوت.

### 4. تشغيل البوت

```bash
# تشغيل عادي
npm start

# تشغيل مع إعادة تحميل تلقائية عند التعديل
npm run dev
```

عند أول تشغيل ناجح ستشوف في الـ Console:
```
[SUCCESS] البوت متصل بنجاح
[INFO] تم تحميل XX أمر
[DB] ✅ MongoDB متصل بنجاح  ← (إن وضعت MONGO_URI)
```

## 💾 قاعدة البيانات (MongoDB) — اختياري

البوت يشتغل **بدون** قاعدة بيانات، لكن إن ربطته بـ MongoDB يحصل على:

- **حفظ بيانات المستخدمين** — XP، المستوى، الرصيد، عدد الرسائل
- **ذاكرة محادثة AI** — يتذكر سياق المحادثة مع Gemini/Cerebras
- **إعدادات المجموعات** — رسالة ترحيب، حماية ضد السبام، prefix مخصص

بدون MongoDB كل شيء يُحفظ في الذاكرة المؤقتة ويُمسح عند إعادة التشغيل.

## ➕ إضافة أمر جديد

أنشئ ملف `commands/mycommand.js` بهذا الهيكل:

```js
module.exports = {
  config: {
    name: "mycommand",       // اسم الأمر
    aliases: ["alias1"],     // أسماء بديلة
    role: 0,                 // 0 = للجميع، 1 = مشرفين فقط
    countDown: 5,            // cooldown بالثواني
    category: "أدوات",
    description: "وصف أمري الجديد",
  },

  onStart: async ({ api, event, args }) => {
    const { threadID, messageID } = event;
    // args = باقي الكلمات بعد الأمر
    // مثال: ".mycommand مرحبا" → args = ["مرحبا"]

    await api.sendMessage("مرحباً! الأمر شغال ✅", threadID, messageID);
  }
};
```

البوت يحمّل الأوامر تلقائياً من مجلد `commands/` عند كل تشغيل.

## 🔧 استكشاف الأخطاء

| المشكلة | الحل |
|---------|------|
| البوت لا يتصل بفيسبوك | تأكد من صحة `FB_EMAIL` و `FB_PASSWORD` في `.env` |
| أوامر AI لا تعمل | تأكد من وضع مفتاح API الصحيح في `.env` |
| MongoDB لا يتصل | تأكد من صحة `MONGO_URI` — البوت يكمل بدونها |
| أمر لا يُحمَّل | تحقق من console عند التشغيل لرسائل `[WARN]` |

## 📝 ملاحظات مهمة

- ⚠️ **لا ترفع ملف `.env`** على GitHub — فيه كلمات مرور ومفاتيح سرية
- استخدم حساباً فيسبوك **مخصصاً للبوت**، وليس حسابك الشخصي
- فيسبوك قد يحظر الحساب إن استخدم البوت بكثافة — هذا خارج عن إرادتنا
- الـ Prefix الافتراضي هو النقطة `.` — يمكن تغييره من `config.json`
